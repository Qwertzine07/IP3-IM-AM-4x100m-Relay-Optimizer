# IP3-IM-AM-4x100m-Relay-Optimizer
This repository contains a website which people can use to optimize their 4x100m running time. This website takes the input of people's velocity and time during a 100m race and then uses both cubic regression analysis and the analysis of the Keller Model to find trends and determine the best arrangement of runners.

The Relay-Optimiser.html file contains the HTML code of the website if you want to run it on an IDE. Here is a link which allows you to access the website directly on your browser: file:///C:/Users/eesha/Downloads/relay_optimizer_v3_6.html

There are many tabs in this which you can explore. The main tab you need is the overview tab to see which order of runners is the best and the add runner data tab where you have to add your own runner's velocity and time.

I hope you all enjoy using this tool!
